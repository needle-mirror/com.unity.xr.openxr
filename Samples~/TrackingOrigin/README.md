# Tracking Origin Sample

This example demonstrates the different behavior of different tracking origin modes, as well as the parameters that affect how [XR origin](https://docs.unity3d.com/Manual/xr-origin.html) is constructed by the application.

For documentation, refer to: https://docs.unity3d.com/Packages/com.unity.xr.openxr@latest?subfolder=samples/tracking-origin.html
